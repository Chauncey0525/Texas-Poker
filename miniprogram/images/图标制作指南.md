# TabBar图标制作指南

## 需要的图标文件

需要在 `miniprogram/images/` 目录下创建以下图标文件：

### 图标列表
1. `home.png` / `home-active.png` - 首页图标
2. `simulation.png` / `simulation-active.png` - 模拟练习图标
3. `replay.png` / `replay-active.png` - 复盘分析图标
4. `knowledge.png` / `knowledge-active.png` - 知识库图标
5. `profile.png` / `profile-active.png` - 个人中心图标

## 图标规格要求

- **尺寸**：81px × 81px（推荐）
- **格式**：PNG（支持透明背景）
- **颜色**：
  - 普通状态：灰色（#7A7E83）
  - 激活状态：主题色（#0f3460）

## 快速制作方法

### 方法一：使用在线图标生成工具

1. 访问 [IconFont](https://www.iconfont.cn/) 或 [Flaticon](https://www.flaticon.com/)
2. 搜索相关图标（如：home, game, replay, book, user）
3. 下载PNG格式，尺寸选择81×81
4. 使用图片编辑工具调整颜色：
   - 普通状态：改为灰色
   - 激活状态：改为主题色（#0f3460）

### 方法二：使用Emoji作为临时图标

可以使用Emoji字符作为临时图标：
- 首页：🏠
- 模拟：🎮
- 复盘：📊
- 知识库：📚
- 我的：👤

但需要将Emoji转换为PNG图片。

### 方法三：使用简单图形

使用简单的几何图形作为图标：
- 首页：方形或房子轮廓
- 模拟：游戏手柄或棋盘
- 复盘：图表或时钟
- 知识库：书本或文档
- 我的：圆形或人形

## 图标设计建议

1. **简洁明了**：图标应该一眼就能识别功能
2. **风格统一**：所有图标使用相同的设计风格
3. **对比度**：确保在白色背景上清晰可见
4. **尺寸适中**：图标内容不要太小，留出适当边距

## 添加图标后的配置

图标文件准备好后，在 `app.json` 中恢复图标配置：

```json
"tabBar": {
  "list": [
    {
      "pagePath": "pages/index/index",
      "text": "首页",
      "iconPath": "images/home.png",
      "selectedIconPath": "images/home-active.png"
    },
    // ... 其他配置
  ]
}
```

## 临时方案

目前 `app.json` 已暂时移除了图标配置，只使用文字标签，这样可以先让项目运行起来。等图标准备好后再添加即可。
